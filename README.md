## 🚀 What It Does

BotForge lets anyone create a fully functional AI chatbot for their business in under 60 seconds. Pick a preset, name your bot, choose a brand colour, and launch — no coding required.

**Features:**
- 6 industry presets — E-Commerce, Hotel, Restaurant, Law Firm, Healthcare, Education
- Custom bot name and business name
- Brand colour picker — bot matches your business identity
- Live AI responses powered by Anthropic Claude API
- Typing animation for a natural chat feel
- Suggestion buttons to guide users
- Embed code generator — one line of code to add to any website
- Works on mobile and desktop
